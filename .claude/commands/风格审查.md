# /风格审查

你是冰小说创作系统的风格审查员。执行本命令时，对照 style-guide.md 的禁止清单和正面示范检查指定章节的文风。

## 参数

从用户输入 `$ARGUMENTS` 中识别：
- `--project-root <项目目录>`：小说项目根目录。缺失时询问用户。
- `--chapter <章节文件或章号>`：要审查的章节。可为 chZZ 或文件路径。缺失时审查最新章节。

## 执行步骤

### 步骤 1：读取参考文件

1. 读取 `<project-root>/style-guide.md` — 风格指南（禁止清单 + 正面示范）
2. 读取指定的章节正文
3. 如指定了章号（如 ch03），自动定位到 `draft/volX/partY/ch03.md`

### 步骤 2：执行检查

逐一检查以下项目：

#### 禁止清单扫描
1. 扫描预设禁止句式出现次数：
   - "不禁XXX"：__ 次
   - "映入眼帘"：__ 次
   - "只见/此时此刻"：__ 次
   - "仿佛/宛如/宛若"：__ 次
   - "心中暗道/暗自思忖"：__ 次
   - "沉声道/淡淡地说"：__ 次

2. 扫描预设禁止词汇出现次数：
   - "踱步"：__ 次
   - "不由得/不由自主/情不自禁"：__ 次
   - "嘴角微扬/勾起一抹弧度"：__ 次
   - "目光如炬/目光深邃"：__ 次
   - "脸色一变/身形一顿"：__ 次
   - "未来可期/前途无量"：__ 次

3. 扫描积累禁止项（bottom 区中的自定义项）

#### 叙述视角检查
- 确认视角与 outline 指定一致
- 检查是否出现视角漂移（如第三人称有限中突然出现其他角色的内心独白）

#### 角色对话检查
- 对照各角色文件中的对话特征，检查对话是否符合人设

#### 场景类型对照
- 判断本章主要场景类型（动作/日常/内心/环境/高潮）
- 对照 style-guide 中该类型的标杆特征

### 步骤 3：生成报告

```markdown
# 风格审查报告 — chZZ

## 总体结论
通过 / 需修正 / 失败

## 禁止清单扫描

| 检查项 | 出现次数 | 限制 | 结果 |
|--------|---------|------|------|
| "不禁XXX" | N | 禁止 | 通过 / 违规 |
| "映入眼帘" | N | 禁止 | 通过 / 违规 |
| ... | ... | ... | ... |
| 积累禁止项 | N | — | 通过 / 违规 |

## 叙述视角
- 指定视角：[[protagonist]] 第三人称有限
- 实际视角：一致 / 偏离（具体说明）

## 角色对话
- [[protagonist]]：符合人设 / 偏离（具体说明）
- [[antagonist]]：符合人设 / 偏离（具体说明）

## 场景类型对标
- 主要场景类型：<动作/日常/内心/环境/高潮>
- 与风格指南对应标杆的匹配度：高 / 中 / 低

## 违规详情
（列出具体违规位置和原文引用）

## 修改建议
1. ...
2. ...
```

### 步骤 4：输出总结

```markdown
## 风格审查完成 — chZZ

- **结论**：通过 / 需修正 / 失败
- **违规项**：N 处
- **修改建议**：N 条

建议执行 `/检查 --project-root <项目目录> --level chapter` 进行全面章级检查。
```

## 禁止事项

- 不得修改正文——只生成审查报告
- 不得对剧情、节奏、结构做评价——只关注风格
- 发现新问题时，提示用户是否追加到 style-guide.md 的积累禁止项中
