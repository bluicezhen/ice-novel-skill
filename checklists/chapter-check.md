# 章级检查清单

> 每章完成后执行。AI 先跑可计数的事实核查，写入报告；然后作者通读确认主观项。

## 一致性

- [ ] 角色位置与 [[character]] 文件一致
- [ ] 角色能力未超出当前弧光规划
- [ ] 时间线推进合理
- [ ] 未出现未注册的新地名/人名
- [ ] 幕与幕之间衔接连续，无无说明的时空或情绪硬切
- [ ] 幕间出现的时间、地点、人物状态变化，已在正文中明确交代过渡

## 大纲对照

- [ ] 剧情与 outline 本章摘要一致
- [ ] 未违反本章禁区
- [ ] 章末钩子已到位
- [ ] 本章如有「弧光推进」标记，该线变化是否按计划触发

## 节奏

### AI 自动核查（写入检查报告）

| 检查项 | 结果 | 判断 |
|--------|------|------|
| 本章总字数 | ____字 | 2000～4000 内 = 通过，超出 = 标记 |
| 各幕字数 vs 简报预估 | 第1幕____/第2幕____/第3幕____... | 任一幕偏差 >30% = 标记 |
| 末段是否呼应章末钩子要求 | 钩子要求：____ → 末段内容：____ | 不呼应 = 标记 |
| 情绪走向 | 首幕情绪____ → 末幕情绪____ | 偏离简报 = 标记 |
| 最长连续对话 | ____句 | >15 句 = 标记 |
| 禁区扫描 | ____处疑似违规 | 列出具体位置 |

### 作者确认

- [ ] 章末钩子有效——我想知道接下来会怎样
- [ ] 节奏不拖不赶——通读时没有想跳过的段落
- [ ] 本章高潮/转折力道足够
- [ ] AI 标记项已逐一确认（是问题的改正文，不是问题的说明原因）

## 风格

- [ ] 对照 [[style-guide]] 检查禁止清单
- [ ] 叙述视角未漂移
- [ ] 角色对话符合各自画像
- [ ] 本章主要场景类型写法是否符合风格指南对应的场景标杆

## 伏笔

- [ ] 本章埋设的伏笔已从「计划中」移至「已埋设」
- [ ] 应在本章回收的伏笔已移至「已回收」
- [ ] 回收时检查回收方式是否与计划一致
